# Cars By Daniel - Premium Auto Detailing Website

A high-end, multi-page Next.js website for a car detailing business in Lafayette, LA.

## Features

- **5+ Impressive Reactive Elements:**
  1. Animated service cards with hover effects
  2. Scroll-triggered animated counters
  3. Parallax testimonials section
  4. Magnetic CTA button
  5. Interactive before/after slider
  6. Smooth page transitions

- **Multi-Page Structure:**
  - Home (/)
  - Services (/services)
  - Gallery (/gallery)
  - About (/about)
  - Contact (/contact)

- **Tech Stack:**
  - Next.js 14 (App Router)
  - React 18
  - Tailwind CSS
  - Framer Motion
  - Remotion (video)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm build
```
